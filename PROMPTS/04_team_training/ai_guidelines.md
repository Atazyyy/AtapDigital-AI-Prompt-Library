## 📝 Common Use Cases

### Use Case 1: Brainstorming Content Ideas

**Prompt Template:**
```
Saya adalah social media manager untuk [brand].
Target audiens: [description].
Buatkan 10 ide konten untuk [topic/theme] yang engaging
untuk platform [Instagram/TikTok/Twitter].
```

**Example:**
```
Saya adalah social media manager untuk brand skincare.
Target audiens: perempuan 18-30 tahun di Indonesia.
Buatkan 10 ide konten tentang "benefits of vitamin C serum"
yang engaging untuk Instagram.
```

**Tips:**
- Be specific tentang audience dan platform
- Tambahkan constraints (budget, timing, dll)
- Iterate: if first output kurang cocok, refine prompt

---

### Use Case 2: Drafting Caption

**Prompt Template:**
```
Buatkan 3 variasi caption untuk post tentang [topic].
Tone: [professional/casual/funny/informative].
Length: [short/medium/long].
Platform: [Instagram/TikTok/Twitter].
Include: [CTA/hashtags/emoji preferences]
```

**Tips:**
- Always add brand voice specifics
- Review dan edit sebelum posting
- Don't forget to add personal touch

---

### Use Case 3: Riset Kompetitor

**Prompt Template:**
```
Analisis social media presence dari kompetitor [competitor name].
Berikan:
1. Content themes yang paling sering
2. Tone dan voice yang digunakan
3. Engagement strategy
4. Strengths dan weaknesses
5. Opportunities untuk kami
```

**Tips:**
- Provide data dari screenshoot/data competitor
- AI analyze based on what you provide
- Always verify dengan observation langsung

---

### Use Case 4: Creating Reports

**Prompt Template:**
```
Buatkan template laporan monthly performance untuk
social media dengan sections:
- Executive summary
- Key metrics
- Top performing content
- Insights dan recommendations
- Next month plan

Format: [Markdown/table/bullet points]
```

**Tips:**
- Customize template sesuai kebutuhan
- AI generate structure, kamu fill in data

## ⚠️ AI Pitfalls & How to Avoid

### Pitfall 1: Hallucination (AI Ngaco)
**Problem:** AI sometimes generates plausible-sounding but incorrect information.

**Solution:**
```
✓ Always verify facts dengan sumber terpercaya
✓ Cross-check numbers dan statistics
✓ Don't use AI untuk real-time news
✓ When in doubt, ask human expert
```

### Pitfall 2: Generic Output
**Problem:** AI output terasa robotic dan generic.

**Solution:**
```
✓ Add specific context ke prompt
✓ Specify brand voice dan personality
✓ Edit dan personalize output
✓ Add local flavor (Indonesia context)
```

### Pitfall 3: Over-reliance
**Problem:** Menjadi terlalu зависим от AI.

**Solution:**
```
✓ AI adalah assistant, bukan replacement
✓ Maintain critical thinking skills
✓ Develop your own expertise
✓ Use AI untuk enhance, not replace creativity
```

### Pitfall 4: Privacy Issues
**Problem:** Accidentally sharing confidential data.

**Solution:**
```
✓ Don't share client names/details yang sensitive
✓ Don't share financial data
✓ Don't share internal strategy documents
✓ Use anonymized/sanitized versions
```

Sebelum menggunakan AI output, selalu check:

```
□ Fakta sudah diverifikasi?
□ Tone sesuai dengan brand voice?
□ Information up-to-date?
□ Tidak ada sensitive data yang ter-expose?
□ Sudah di-edit biar terasa personal?
□ Grammar dan spelling sudah benar?
□ Hashtags sudah relevant?
□ CTA sudah appropriate?
□ Overall feels authentic?
```

## 🆘 Troubleshooting

**Problem:** AI gives irrelevant answer
```
→ Refine your prompt with more context
→ Break down your question into smaller parts
→ Try different wording
```

**Problem:** AI output seems biased
```
→ AI reflects training data biases
→ Be aware of potential biases
→ Always apply critical thinking
→ Seek diverse perspectives
```

**Problem:** AI tool is down
```
→ Have backup manual process ready
→ Don't rely on single tool
→ Build internal knowledge base
```
